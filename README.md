# 📚 LitShelf API (Bookstore Management System)

A Spring Boot REST API for managing a bookstore with authentication and CRUD operations.

## 🚀 Features
- 🔐 JWT Authentication (Login/Register)
- 📖 Book Management (Add, Update, Delete, View)
- 👤 User Management
- 📦 RESTful API design
- 🗄 MySQL Database integration

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- MySQL
- Maven

## 📂 Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- DTOs & Entities
- Security Config (JWT)

## ▶️ How to Run
1. Clone the repo
2. Configure MySQL in `application.properties`
3. Run the Spring Boot application

## 📌 API Endpoints
- `/auth/register`
- `/auth/login`
- `/books`
