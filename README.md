# 📚 Bookstore API (Spring Boot)

A RESTful backend application built using Spring Boot.

## 🔧 Features
- User Registration & Login (JWT Authentication)
- Book Management (CRUD APIs)
- Secure endpoints using Spring Security
- Global Exception Handling

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- Maven

## 🚀 How to Run
1. Clone the repo
2. Configure MySQL in application.properties
3. Run Spring Boot application

## 📌 API Endpoints
- POST /auth/register
- POST /auth/login
- GET /books
- POST /books
- PUT /books/{id}
- DELETE /books/{id}
